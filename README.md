# DesignPatterns

1) read string \n
2) read subject
 a - create Node for subject
 b - set Node in NodeMap
3) read predicate
 a - create Predicate for predicate
 b - set Predicate in PredicateMap
4) read object
 a - create Node for subject
 b - set Node in NodeMap
5) set subject, predicate, object on Triple
 a - add to TripleMap
6) add triple to queryMapSet
